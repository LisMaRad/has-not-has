• :not(:has()) matches if none of the child element has a div element.

  • :has(:not()) matches if one of the child element doesn't have a p element.
  • Source: Reddit

Based on the website Can-i-use and Mozilla Developer the :has() and :not() selectors are supported in the main browsers, especially the desktop browser versions. Therefore, using the selectors is very convenient to check for classes or child elements and style certain child elements.

Based on the website Bram the support can be checked by using @supports selector(:has(*))

Fallback:

As a fallback, it is possible to check the classes and child elements with Javascript. Therefore a function has to query the element and add the styling afterwards.
